# Sentiment Analysis Web App (IBM Watson + Flask)

## Overview
This project is a **sentiment analysis web application** built using **IBM Watson AI libraries** and the **Flask framework**. The application analyzes user-provided text and determines sentiment in real time through a simple web interface.

The project demonstrates how AI services can be integrated into a production-style web application and deployed as a web service.

## Features
- Real-time sentiment analysis
- Integration with IBM Watson AI libraries
- Flask-based backend
- Interactive web interface
- Deployable as a web service

## Tech Stack
- Python
- Flask
- IBM Watson AI Libraries
- HTML/CSS
- REST API Integration

## How It Works
1. The user enters text into the web interface.
2. The text is sent to the Flask backend.
3. The backend sends the request to the IBM Watson sentiment analysis service.
4. The sentiment result is returned and displayed to the user.
